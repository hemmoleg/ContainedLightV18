To add a movie:
  - open assets/movies.json
  - make necessary changes
  - open routes.txt
  - add new line /movies/<moveID> (eg. movies/radiostar)
  - save all
  - run ng build
  - copy new files to server

  To remove a movie:
  - open assets/movies.json
  - make necessary changes
  - open routes.txt
  - remove line /movies/<removedMovieID>
  - save all
  - run ng build
  - copy new files to server

To change a movie:
  - open assets/movies.json
  - make necessary changes
  - save
  - run ng build
  - copy new files to server
